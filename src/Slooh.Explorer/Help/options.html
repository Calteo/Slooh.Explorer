Dashboard

The download is controlled by the options in the upper portion.

Options

Folder

This is the base folder where all the output is saved. Unless one of the other options contains an absolute path (i.e. "D:\Slooh\Images"). With the "Select" button you can pick and create a folder with a dialog.

Mission

  • The contains the path that is appended to the folder option.
  • It can contain placeholders to put every mission into a different sub folder.
  • It gets prepended too all of the following options.
  • Example: ${title}\${date:yyyy-MM-dd}\${date:HH-mm-ss}

Information

  • An information file about the mission is saved.
  • It is only saved if the option is checked.
  • It can contain placeholders.
  • Checking the "overwrite" option will overwrite an existing file.
  • The format can be picked a the dropdown to the right.
  • Example: ${telescope}-Information

Information

Picture

  • The images of the mission are saved.
  • It is only saved if the option is checked.
  • It can contain placeholders.
  • Checking the "overwrite" option will overwrite an existing file.
  • Example: ${telescope}\${instrument}\${filename}

JPEG

  • The images of the mission are saved in JPEG format.
  • It is only saved if the option is checked.
  • It can contain placeholders.
  • Checking the "overwrite" option will overwrite an existing file.
  • Example: D:\temp\SloohJpeg\${title}\${filename}. This puts the JPEG not under the Folder option, but rather fixed in D:\temp\SloohJpeg

FITS

  • The FITS files of the mission are saved.
  • It is only saved if the option is checked.
  • It can contain placeholders.
  • Checking the "overwrite" option will overwrite an existing file.
  • Example: FITS\${telescope}\${instrument}\${filename}

Placeholder

Placeholders will insert attributes of the mission/image into the filename to use for saving the data. All placeholder much be in the format ${name} or ${name:format}.

Name Description
title The title of the mission/picture.
date The date of the mission/picture. You can add a format hear (i.e. yyyy-MM-dd) to define how the date is converted into a text. The default is yyyy-MM-dd-HH-mm-ss
telescope The telescope of the mission/picture.
instrument The instument of the mission/picture.
filename The filename of the file to save. This should be included in the Picture, JPEG and FITS options. Otherwise the files will get saved all under the same name. For the Mission option this is not available.
