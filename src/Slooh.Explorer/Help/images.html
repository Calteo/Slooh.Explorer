Dashboard

This gives you the overview of your images and sets the options fro download. The list of images is populated automatically once you logged on.

Images

Column Description
Date The date the mission took place.
Title The title of the mission.
P The state of the process. Shows if you already downloaded the images or not.
Image The number of images in this mission.
FITS Are FITS files in this mission.
Telescope Name of the telescope capturing this mission.
Instrument Name of the instrument capturing this mission.
State Status during the download process.

In the green fields below the captions you can enter a filter expression, so the list gets reduced to the matching items.

  • Entering some text will find all missions containing that text (i.e. "M105" will match the title "Elliptical Galaxy M105").
  • Entering "*" will match any characters and "?" will only match a single character.
  • On the text columns there can be an operator at the beginning of the filter expression. Possible are "=", "<=", ">=", ">", "<" and operate like they normally do.
  • On the Date column there are only ">=", "<=", "<" and ">". Entering just a date will match all missions on that date.

Download

Mark some missions and press to Download button to start downloading the images. Once they are downloaded the thumbnails are shown on the right hand side. Double clicking on a thumbnail opens the image in the standard windows viewing programm.

Deleting

If you selected some missions you can open the context menu. If allows you to open the local folder in a explorer window. Also you can delete the mission from the slooh web site. The local files are kept if the were already downloaded. You can also mark the thumbnails by pressing the ctrl-key while left clicking on a thumbnail. The frame around the thumbnail changes. Open the context menu on the thumbnails and delete only certains images. The local files are kept if they were already downloaded.
